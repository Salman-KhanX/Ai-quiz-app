# 🤖 AI Quiz App

An interactive AI Quiz App built using **HTML**, **CSS**, and **JavaScript**. This project tests users' basic knowledge of Artificial Intelligence through multiple-choice questions and provides instant feedback with a final score.

---

## ✨ Features

* 📝 10 AI Multiple Choice Questions (MCQs)
* ✅ Instant Correct & Wrong Answer Feedback
* 📊 Automatic Score Calculation
* 🔄 Restart Quiz Functionality
* 🎨 Clean and Responsive User Interface
* ⚡ Built with Vanilla JavaScript

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

---

## 📂 Project Structure

```text
AI-Quiz-App/
│
├── index.html
├── style.css
└── script.js
```

---

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **index.html** or open it in your browser.
4. Start the quiz and check your score.

---

## 🎯 Project Objective

The goal of this project is to practice JavaScript fundamentals, including DOM manipulation, event handling, conditional statements, arrays, and interactive user interfaces.
